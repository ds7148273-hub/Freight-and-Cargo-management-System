# Freight And Cargo Management System

Modern React + Vite project for a freight and cargo management landing experience.

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

The app is configured to run on `http://127.0.0.1:5173`.

## Build

```bash
npm run build
```
